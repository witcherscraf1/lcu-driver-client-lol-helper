README

Ban/Pick规则讲解：
所谓Ban就是禁止使用，Pick就是选取的意思。也就是说，在比赛开始前，双方要根据己方或对方的战略战术Ban掉一些英雄来确定选取的英雄。

bp顺序：
禁用英雄环节:蓝方4楼-红方4楼-蓝方5楼-红方5楼
选英雄环节:蓝方1楼-红方1.2楼-蓝方2.3楼-红方3.4楼-蓝方4.5楼-红方5楼

程序运行详情
1.运行Satrt.py

2.用户选择红/蓝方

3.用户输入15个英雄组成备选英雄池，例如：

狂铁
梦奇
铠
曜
裴擒虎
赵云
杨玉环
女娲
干将莫邪
李元芳
后羿
狄仁杰
盾山
鲁班大师
瑶

4.程序提供建议ban掉的英雄，用户可以输入建议ban选的英雄，也可输入其他

5.程序提供目前阵容下建议选择的英雄，用户输入选择的英雄

6.程序最后输出我方和敌方的队伍阵容
